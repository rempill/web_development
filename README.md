# 🌐 Web Development Course Repository

**Year 2 – Semester 2**

## 📌 Overview

This repository contains coursework, labs, and projects developed during the **Web Development** course in Year 2, Semester 2. It covers essential concepts for building modern web applications, including frontend development, backend systems, client-server communication, and security practices.

---

## 🎯 Course Objectives

* Understand the structure and standards of web technologies
* Build responsive and interactive user interfaces
* Develop server-side applications and connect them to databases
* Learn core web protocols and client-server communication
* Apply best practices in web security and SEO

---

## 📚 Course Content

### 🧱 HTML

* Structure of an HTML document
* Basic tags and elements
* Differences between HTML 4.01 and HTML5
* Web standards

### 📝 Web Forms

* Creating and handling forms
* Input types and validation

### 🎨 CSS & Responsive Design

* Cascading Style Sheets (CSS) fundamentals
* Layout techniques
* Responsive Web Design (RWD)

### 🌍 HTTP Protocol

* HTTP basics
* GET and POST methods
* Stateless nature of HTTP
* Cookies and session management

### ⚡ JavaScript & DOM

* JavaScript fundamentals
* JSON format
* Document Object Model (DOM) manipulation

### 🔄 AJAX & Frameworks

* Asynchronous requests with AJAX
* jQuery basics
* Client-side frameworks:

  * Bootstrap
  * AngularJS

### 🖥️ Backend Development (LAMP Stack)

* Linux, Apache, MySQL, PHP
* PHP fundamentals
* Database interaction using MySQL
* Apache server basics

### 🔗 Web Services

* Introduction to web services
* API communication

### 🔐 Web Security

* SQL Injection
* Cross-Site Scripting (XSS)
* Cross-Site Request Forgery (CSRF)
* Unrestricted file upload vulnerabilities

### ☕ Java Web Technologies

* Java Servlets
* JSP (JavaServer Pages)
* JSTL
* Custom tags
* JavaServer Faces (JSF)

### 🧩 Content Management Systems

* Overview of CMS platforms
* Usage and customization

### 📈 SEO (Search Engine Optimization)

* SEO basics
* Best practices for improving visibility

---

The repository is organized into multiple lab folders (e.g., `lab1`, `lab2`, ...), each focusing on specific topics covered during the course.
